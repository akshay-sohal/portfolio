# Portfolio Website

This is my **responsive portfolio website**, built using **React, HTML, and CSS**. It showcases my projects, skills, and contact information.

## Features
- Fully **responsive design** for desktop, tablet, and mobile
- Smooth navigation and user-friendly layout
- Portfolio section with project showcases
- Contact form for inquiries

## Technologies Used
- React
- HTML5
- CSS3 (Flexbox & Grid)
